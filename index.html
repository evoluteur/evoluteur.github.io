Me

My name is Olivier Giulieri (username Evoluteur). I live in the San Francisco Bay Area with my lovely cat Neige. I work in Tech. I'm French.

This page is a more fun and personal way to introduce myself than my LinkedIn and GitHub profiles.

Code

I like to code (mostly Javascript and CSS, sometimes SQL). I usually build user interfaces for startups.

My favorite quote about programming is "Smart data structures and dumb code works a lot better than the other way around." - Eric S. Raymond in The Cathedral and the Bazaar.

I am very interested in metadata and generic code (usually found in model-driven architecture, low-code platforms, and data visualization...). I experiment with my open-source project Evolutility model-driven UI and backend.

Cooking

I like to cook (and eat), especially desserts. My specialties are crèmes brûlées, pear cream tart, and madeleines. I'm learning to make cannelés.

I recently started experimenting with salty madeleines. Salty madeleines with duck prosciutto are really good, but I still prefer the sweet madeleines with lemon zest and pine nuts.

Inspiration
